# Product Knowledge

> 最近更新：2026-06-05

这里保存月点的长期产品定位、愿景和产品原则。它回答“月点最终想成为什么”，但不直接等同于当前比赛 MVP 的交付承诺。

## 当前入口

- `vision.md`：月点长期产品愿景，以及它和当前提交范围的边界。

## 维护规则

- 产品愿景可以表达长期 IDE 方向，但必须明确哪些不属于当前 MVP。
- 已确认会影响近期开发顺序的内容，应同步进入 `../../planning/roadmap/`。
- 仍处于发散状态的想法先留在 `../brainstorm/`。
- 已稳定的数据模型、导出契约和编辑语义分别沉淀到 `../architecture/`、`../schema/`、`../interaction/`。
